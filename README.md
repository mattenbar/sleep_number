# Sleep Number PDP Gallery Demo

A small static demo that shows how product selection can swap a Cloudinary Product Gallery media set for different Sleep Number variants.

## Features
- Interactive color selector for different product variants
- Cloudinary-powered image gallery updates
- No build step required; open the files directly or serve the folder locally

## Run locally

Option 1: open index.html in a browser.

Option 2: serve the folder with a simple HTTP server:

```sh
python3 -m http.server 5173
```

Then visit http://localhost:5173/.

## Notes
- The demo uses the public Cloudinary asset list for the Sleep Number gallery.
- No secrets or private credentials are included in this project.
